# DevMate Project Overview

DevMate is an AI-powered onboarding assistant designed to help new developers join the team efficiently.

## Core Features
1. **RAG-based Q&A**: Answers questions using project documentation.
2. **Role-based Context**: Tailors answers for Interns, Juniors, or Seniors.
3. **Automated Setup**: Guides users through installation.

## Architecture
- **Frontend**: Streamlit
- **Backend**: Python, LangChain
- **LLM**: Amazon Nova Lite via AWS Bedrock
- **Vector Store**: FAISS
